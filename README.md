
# Project Title

Sample React app, with unit tests and e2e tests.




## Features

- Display a list of users
- Sort list
- Add new users


## Run project

To run this project

```bash
    - Clone repo
    git clone https://github.com/mz4/users.git .
    
    - Install dependencies
    yarn
    
    - Start local server
    yarn start

    - Run unit tests
    yarn test

    - Run Cypress tests
    yarn cypress:open
```
